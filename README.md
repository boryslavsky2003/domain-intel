# Domain Intel

Інструмент для автоматичного аналізу та оцінки доменних імен з використанням GoDaddy API.
Побудований на принципах **Clean Architecture** (Domain, Application, Infrastructure, Presentation).

## Як це працює

Програма виконує наступні кроки для кожного домену:
1.  **Перевірка доступності**: Чи вільний домен для реєстрації?
2.  **Оцінка вартості (GoValue)**: Отримання прогнозованої вартості та ймовірності продажу.
3.  **Аналіз (Business Rules)**:
    *   Домен має бути вільним.
    *   Оціночна вартість (GoValue) >= $500.
    *   Ймовірність продажу >= 20%.
4.  **Рекомендація**: `BUY` (Купувати) або `SKIP` (Пропустити).

## Налаштування

1.  **Створіть віртуальне середовище:**
    ```bash
    python -m venv venv
    source venv/bin/activate
    ```

2.  **Встановіть залежності:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Налаштуйте ключі доступу:**
    Скопіюйте приклад файлу конфігурації:
    ```bash
    cp .env.example .env
    ```
    Відкрийте `.env` та вставте ваші API ключі GoDaddy ([отримати тут](https://developer.godaddy.com/keys)).

## Використання

### CLI (Командний рядок)
Запустіть скрипт, передавши список доменів через пробіл:

```bash
python main.py example.com mydomain.com google.com
```

### TUI (Текстовий інтерфейс)
Для запуску інтерактивного режиму з таблицею:

```bash
python main.py tui
```
